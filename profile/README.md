## 주요프로젝트
### DAEXI
#### 소개
혼자 택시를 타기에는 부담스럽고, 대중교통은 너무 복잡할 때!
**DAEXI**는 실시간 위치 기반 매칭과 목적지 유사도 분석을 통해 택시를 함께 탈 수 있는 사용자를 찾아줍니다.
이를 통해 사용자들은 비용을 절감하고, 교통 효율도 높일 수 있습니다.

#### 주요 기능

- 출발지/목적지 입력을 통한 그룹 생성 및 관리
- 택시비 자동 1/n 정산 기능
- 결제하는 사람의 계좌 번호 표시
- 약속 장소, 시간 정하기
- 만나기전 조율을 위한 채팅 기능
